## Quick start
1. Copy the project files into a new folder.
2. Put your Figma-exported `Home.svg` into `src/assets/Home.svg`.
3. Install dependencies:


```bash
npm install
```


4. Start dev server:


```bash
npm run dev
```


Open the dev URL printed by Vite (usually http://localhost:5173).
